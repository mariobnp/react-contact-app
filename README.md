# React Contact App

A simple contact management application built with **React** that allows users to register, login, and manage their contacts through an API.

## 📚 About This Project

This project was developed as part of my learning journey in the **React course from Coding Camp 2026**.

The application was built based on a study case provided in the course.
I implemented the required features while also exploring React concepts such as:

* Component-based architecture
* State management
* API integration
* Authentication flow
* CRUD operations

## ✨ Features

* User Registration
* User Login
* Add Contact
* Delete Contact
* Fetch Contact Data from API

## 🛠️ Tech Stack

* React
* JavaScript
* REST API
* HTML
* CSS
